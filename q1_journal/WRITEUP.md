# Q1 — Trading journal / performance system

## Purpose

Prove **self-directed trading discipline**: rules, risk sizing, and review metrics — not hype screenshots.

## What’s in this folder

| File | Role |
|------|------|
| `sample_trades.csv` | Demo journal rows (OB/MS setups, R-multiples, notes) |
| `run_journal_metrics.py` | Computes win rate, expectancy, max drawdown in R, daily net R |
| `journal_report.html` | Generated presentable report (run the script) |

## How James uses this in real life

- Journal every trade with setup tag, risk %, R result, and post-trade note  
- Daily review: net R, whether max daily loss rule fired  
- Replace `sample_trades.csv` with an anonymized export from the journal app / MT5 history  

## Honesty

Sample CSV is **illustrative structure** for the portfolio demo. Live account stats stay private unless you choose to share aggregates later.
